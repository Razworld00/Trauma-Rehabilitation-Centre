# Assets Directory

Store all media used by the website in this folder:

- `logos/` official logo files (SVG/PNG/WebP)
- `images/` page images (hero, gallery, cards, backgrounds)
- `media/` videos, audio, and downloadable files

Recommended naming:

- `logos/trc-logo-primary.svg`
- `images/hero-eastern-cape-classroom.webp`
- `media/intro-video.mp4`

Use optimized formats where possible:

- Logos: `SVG` (preferred), fallback `PNG`
- Photos: `WebP` or optimized `JPG`
- Videos: `MP4 (H.264)` for broad browser support
